¿Que es HTML?

es el lenguaje estándar utilizado para crear y diseñar páginas web. Se basa en un conjunto de elementos o "etiquetas" que estructuran el contenido, como texto, imágenes, enlaces, vídeos y formularios.

¿Que es CSS?

es un lenguaje de diseño utilizado para describir la presentación de documentos escritos en HTML o XML. Mientras que HTML se ocupa de la estructura y el contenido de una página web, CSS se encarga de cómo se ve ese contenido.

¿Que es Javascript

JavaScript es un lenguaje de programación que se utiliza principalmente para crear interactividad en las páginas web. Es uno de los tres pilares fundamentales del desarrollo web, junto con HTML y CSS.
